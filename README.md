# MyRetail Example

To run:

1. `git clone https://github.com/outdooricon/my-retail-example.git`
2. `cd my-retail-example`
3. `npm i`
4. `npm run start:production`
5. [http://localhost:3000](http://localhost:3000)

To run test suite:
- `npm test`
